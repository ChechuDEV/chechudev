## The challenge

Racing simulation centers often operate multiple Assetto Corsa stations, each
requiring manual configuration and management. This process is time-consuming
and prone to errors, especially during events with multiple sessions. The
clients needed a solution that centralized the control of all stations, allowing
for synchronized game sessions, real-time monitoring, and efficient
configuration management from a single dashboard.

## The architecture

Simcenter is designed as a distributed system with a central brain and multiple
edge agents.

**1. Central brain (Go & Svelte):** A central server manages the overall system,
handling configuration deployment, session control, and real-time monitoring. A
web-based dashboard provides a user-friendly interface for controlling the
system, allowing operators to launch sessions, monitor station status, and
manage configurations.

**2. Edge agents (Go):** Each simulation station runs a local agent responsible
for applying configurations, managing the Assetto Corsa instance, and reporting
status back to the central server.

## Key Technical Hurdles

**1. Real-time synchronization:** Ensuring that the dashboard accurately
reflects the status of all stations in real-time without saturating the network
with excessive updates was a real challenge. We implemented Server-Sent Events
(SSE) to push updates from the central server to the dashboard efficiently, and
WebSockets for communication between the central server and edge agents.

**2. Operational simplicity:** The system needed to be easy to use for operators
who may not be technically savvy. We focused on creating an intuitive dashboard
with clear controls and visual feedback, ensuring that operators could manage
sessions without needing to understand the underlying technical details.

## The impact

**- Reduced Prep Time:** Significantly reduced the time required to prepare for
racing sessions, allowing to create a group and launch a session in seconds
instead of minutes.

**- Improved User Experience:** Enhanced the overall user experience for both
operators and racers, providing a seamless and efficient way to manage multiple
simulators.

**- Scalability:** The system is designed to scale easily, allowing for the
addition of more stations without significant changes to the architecture.

Simcenter has become an essential tool for many racing simulation centers,
streamlining operations and enhancing the experience for both operators and
racers.
